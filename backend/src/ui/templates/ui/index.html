How This Dashboard Works

Start here if this is your first simulation.

This interface lets you watch how a simulated agent reacts to situations over time. You do not need to understand the model first. Start one run, add a few steps, then use the summaries and charts to understand what changed.

1
Create a run
Use Start Simulation to load a personality profile, available actions, and the initial settings.
2
Choose your exploration mode
Use Manual Step for direct control, or AI-Assisted Step if you want the system to propose the next situation.
3
Read the trajectory
Track mood, energy, frustration, actions, and emotions to see whether the agent is becoming stable, reactive, or exploratory.
4
Ask for intervention later
Use Intervention after a few steps when you want the AI to recommend how to steer the run.
